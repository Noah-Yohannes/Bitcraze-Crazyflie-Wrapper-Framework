# Sample Flights

This directory contains example Crazyflie flight demonstrations executed using the Bitcraze Wrapper Framework.

In all demonstrations, the drone flies *from right to left relative to the viewer's perspective*.

## Flight Scenarios

* **Straight line flight** — The drone travels in a simple straight trajectory between two waypoints.
* **Bell curve shape flight** — The drone follows a smooth bell-shaped path, rising and descending along the trajectory.
* **Wobbly bell curve flight variation** — A bell-shaped trajectory with small oscillations that introduce a slight wobble along the path.
* **Deviating flight variation** — The drone deviates from the nominal trajectory before returning to the expected path.
